This means you can now **seamlessly manage and deploy updates for Tomcat 11** across your endpoints using Easy2Patch — just like you already do with other critical applications.

Why This Matters

Apache Tomcat is a widely-used open-source Java servlet container and web server. Version 11 introduces several improvements, modern Java compatibility, and enhanced security — making timely updates more important than ever.

Traditionally, updating third-party apps like Tomcat requires manual effort, scripting, or external tools. With Easy2Patch, it’s now **fully automated and integrated** into your **Microsoft Intune or SCCM** environment.

Benefits of Managing Tomcat 11 with Easy2Patch

- **Automated Updates**: Schedule and deploy updates silently across your systems.
- **Security Compliance**: Ensure all endpoints run the latest secure version of Tomcat.
- **Less Manual Work**: No more downloading and packaging manually.
- **Integration with Intune & SCCM**: Unified update management through your existing infrastructure.

If you're already an Easy2Patch user, simply:

1. Go to the Easy2Patch Application, Update Or Intune Pages.
2. Search for **Apache Tomcat 11**.
3. Select the version you want.
4. Let Easy2Patch handle the deployment via Intune, SCCM or WSUS.
