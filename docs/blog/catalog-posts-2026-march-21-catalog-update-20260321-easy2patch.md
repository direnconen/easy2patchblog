Reference: AIR-7924 Title: AIR files need to have the correct CRC for the hash file Applies to: Core build tools Document Id: HCS19-000287 Description: An .air file is actually a zip file, and as part of the validation/verification there is a hash of part of the file which is calculated and then injected back into the zip. The CRC value for this was only being updated in one of the locations, which was causing a problem for some zip-based applications. This has been updated so that the .air files are completely valid .zip files now.
