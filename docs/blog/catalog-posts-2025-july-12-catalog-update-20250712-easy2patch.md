> Release v1.16.1 / 5.71.1 Release Notes With this release, Sandboxie’s core has been modernized to ensure long-term stability and compatibility with today’s development platforms. We’ve upgraded our build environment to the MSVC 2022 toolset—an unavoidable change now that support for the 2019 runner has been discontinued—and migrated our UI framework from Qt 5 to Qt 6.
